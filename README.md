# 🌍 Nomad Voyage — Morocco Travel Agency Website

A responsive Morocco travel agency website built with HTML & CSS

> 🧑‍💻 Built by me as one of my first responsive website projects — focused on practicing HTML, CSS, Flexbox, Grid, and media queries.

---

## 📸 Preview

> A fully responsive, single-page travel website built with pure HTML & CSS — no frameworks, no JavaScript dependencies.
> This is one of my first projects focused on mastering responsive design.

---

## 🗺️ Sections

- **Hero / Header** — Full-screen background with a booking CTA
- **Destinations** — Cards showcasing Chefchaouen, Merzouga, Ouzoud Waterfalls, and Marrakech
- **Transportation** — Info about the Ghazala bus partnership
- **Gallery** — Grid layout of trip photos
- **Feedbacks** — Customer testimonials with star ratings
- **Booking Form** — Contact form to book a trip
- **Footer** — Social media links (Email, Instagram, Facebook, WhatsApp, YouTube)

---

## 🛠️ Built With

- **HTML5**
- **CSS3** (Flexbox, CSS Grid, Media Queries)
- [Bebas Neue](https://fonts.google.com/specimen/Bebas+Neue) — Google Fonts
- [Font Awesome 4.7](https://fontawesome.com/v4/) — Icons

## 📱 Responsive Design

The site is fully responsive across three breakpoints:

| Breakpoint | Target |
|---|---|
| `max-width: 768px` | Mobile phones |
| `768px – 992px` | Tablets |
| `min-width: 992px` | Desktops |

---

## 🚀 Getting Started

No installation or build step required. Just clone and open:

```bash
git clone https://github.com/saad-ait-elhammoumia/responsive-travel-site.git
cd responsive-travel-site
```

Then open `index.html` in your browser — or use a local server like [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) in VS Code.
